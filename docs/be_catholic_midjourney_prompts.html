Como usar

Cole cada prompt diretamente no Discord do Midjourney com /imagine. Os parâmetros --ar definem o aspect ratio. Use --v 6.1 para máxima qualidade. Para wallpaper 9:16 use --ar 9:16. Para feed 1:1 use --ar 1:1. Para banner use --ar 16:9.

Dica de ouro

Para adicionar as headlines depois, gere os KVs sem texto primeiro — as fontes do Midjourney raramente ficam boas. Use o arquivo Figma/Canva/Photoshop para aplicar a tipografia Cinzel por cima da imagem gerada. Os prompts abaixo já omitem texto propositalmente.

Dicas Gerais
Parâmetros Essenciais
Aspect Ratios

Wallpaper/Stories: --ar 9:16
Feed Instagram: --ar 1:1
Banner horizontal: --ar 16:9
Banner vertical: --ar 2:3

Qualidade & Versão

Sempre use --v 6.1 para máxima qualidade fotorrealista. Adicione --q 2 para maior detalhe. Use --style raw quando quiser menos "filtro" do Midjourney.

Negativo (--no)

Adicione ao final: --no text, watermark, logo, cartoon, anime, distorted faces, ugly, low quality para evitar resultados ruins.

Usar suas referências

Use --sref [URL da imagem] para que o Midjourney imite o estilo visual da sua referência. Cole a URL da imagem depois de fazer upload no Discord.

Seed consistente

Quando gostar de um resultado, copie o --seed XXXXXX que aparece no job para regenerar variações consistentes com o mesmo estilo.

Texto é separado

Nunca peça texto/headline no Midjourney. Gere a imagem limpa e aplique a tipografia Cinzel no Figma, Canva ou Photoshop por cima.
